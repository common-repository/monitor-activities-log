== Description ==

Get notified with all plugin activities inside your applicatin. In one place you can track plugin activities by the users such as activation, deactivation, installation, deletion and more.

= Features =
  &#9989; List all plugin's activities<br>
  &#9989; Filter data by Plugin Name, Plugin Status, Users, Users Caps, Date<br>


== Installation ==

1. Upload `monitor-activities-log` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to 'Plugin Activities' Menu - {domain.com}/wp-admin/admin.php?page=mpa_main_menu

== Frequently Asked Questions ==

= How to see all plugin activities =

Navigate to 'Plugin Activities' Menu in the wordpress backend - {domain.com}/wp-admin/admin.php?page=mpa_main_menu

= How to filter the list =

You will find a few dropdown and input options above the list. Select them as you required and hit the 'Filter' Button.
